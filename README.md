# Macroambiente Landing

Landing corporativa de Macroambiente lista para despliegue estatico.

## Despliegue

El sitio principal esta en `index.html`. La version desplegable queda autocontenida:
estilos, scripts y logos principales estan embebidos para facilitar despliegue en Hostinger.

Para Hostinger, conecta este repositorio y publica la raiz del proyecto:

- Repositorio: `latticeprojectnw-ui/macroambiente-landing`
- Branch: `main`
- Carpeta de publicacion: `/`
- Archivo inicial: `index.html`

No requiere build ni instalacion de dependencias.
